# Setup AWS 

1. Create a new AWS Account and setup using debit card verification 
2. Create two security groups 
   1. Master-SG
   2. Slave-SG
3. Create two instance (EC2)
   1. Kube-Master
   2. Kube-Slave


## Different container orchestration tools : 

1. Docker Swarm
2. Docker Compose
3. Apache Marathon
4. Open Shift
5. K8s (Kubernetes)
